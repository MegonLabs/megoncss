## Usage

### Text Color

```html
<p class="text-white">White text</p>
<p class="text-black">Black text</p>
<p class="text-gray">Gray text</p>
<p class="text-blue">Blue text</p>
<p class="text-red">Red text</p>
<p class="text-green">Green text</p>
```

### Background Color

```html
<div class="bg-white">White background</div>
<div class="bg-black">Black background</div>
<div class="bg-gray">Gray background</div>
<div class="bg-blue">Blue background</div>
<div class="bg-red">Red background</div>
<div class="bg-green">Green background</div>
```

### Border Color

```html
<div class="border border-blue">Blue border</div>
<div class="border border-red">Red border</div>
<div class="border border-gray">Gray border</div>
```

### Shade Variations

```html
<!-- Light backgrounds -->
<div class="bg-blue-50">Lightest blue</div>
<div class="bg-blue-100">Very light blue</div>
<div class="bg-blue-200">Light blue</div>

<!-- Medium -->
<div class="bg-blue-400">Medium blue</div>
<div class="bg-blue">Default blue</div>
<div class="bg-blue-600">Darker blue</div>

<!-- Dark -->
<div class="bg-blue-800">Dark blue</div>
<div class="bg-blue-900">Very dark blue</div>
```

## Examples

### Status Colors

```html
<!-- Success -->
<div class="bg-green-50 text-green-700 pad-16 round-8">
  Success! Your changes were saved.
</div>

<!-- Warning -->
<div class="bg-yellow-50 text-yellow-700 pad-16 round-8">
  Warning! Please check your input.
</div>

<!-- Error -->
<div class="bg-red-50 text-red-700 pad-16 round-8">
  Error! Something went wrong.
</div>

<!-- Info -->
<div class="bg-blue-50 text-blue-700 pad-16 round-8">
  Info: New update available.
</div>
```

### Badge Colors

```html
<span class="badge bg-green text-white">Active</span>
<span class="badge bg-red text-white">Inactive</span>
<span class="badge bg-yellow text-black">Pending</span>
<span class="badge bg-blue text-white">New</span>
```

### Dark Mode

```html
<div class="bg-white dark:bg-gray-900 text-black dark:text-white">
  Adapts to dark mode
</div>
```

### Hover States

```html
<button class="bg-blue hover:bg-blue-700 text-white">Hover me</button>

<a class="text-blue hover:text-blue-700"> Link with hover </a>
```

## Tips

- Use `text-gray` for secondary text
- Use `text-gray-400` for placeholder text
- Use `bg-gray-50` for subtle backgrounds
- Use `bg-gray-100` for card backgrounds
- Use semantic colors: `green` for success, `red` for error
- Use shade variations for hover/focus states
- Combine `text-{color}` with `bg-{color}` for contrast
